# Semaforo Expertise

Descrição
===============
Imagine um semáforo inteligente que atua recalculando o periodo de tempo em que os estagio do sinal são acionados, podendo melhorar o tráfego de uma cidade. 
Possui capacidade de entender o fluxo do trânsito em tempo real, uma vez que utiliza informações das câmeras de monitoramento já existentes nos cruzamentos de algumas vias da cidade.
